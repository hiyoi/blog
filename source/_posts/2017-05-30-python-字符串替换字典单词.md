把text中对应word_dic中的key替换成values

```
def replace_words(text, word_dic):
    yo = re.compile('|'.join(map(re.escape, word_dic)))

    def translate(mat):
        return word_dic[mat.group(0)]
    return yo.sub(translate, text)
```
