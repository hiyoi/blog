匹配重复数字 如
```
111111
222222
333333333
```

可以使用
```
^(\d)\1{5}$
```

(\d)  匹配一位数字
\1    匹配分组1中的内容，即(\d)
{5}   匹配\1五次

匹配AABB数字，如 1122

```
^(\d)\1{1}(\d)\2{1}$
```
